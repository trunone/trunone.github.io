手動編譯 git 原始碼與安裝 (CentOS 5)

使用 Yum 移除及安裝所需套件：
# yum remove curl curl-devel
# yum install zlib zlib-devel libssh2 libssh2-devel openssl openssl-devel expat expat-devel

下載 cURL 原始碼並編譯及安裝：
# cd /usr/local/src/
# wget "http://curl.haxx.se/download/curl-7.21.3.tar.gz"
# tar xzf curl-7.21.3.tar.gz
# rm curl-7.21.3.tar.gz
# ln -s curl-7.21.3 curl
# cd curl/
# make clean
# ./configure --libdir=/usr/lib64 --with-libssh2 --enable-shared --with-ssl --prefix=/usr
# make
# make install

下載 git 原始碼並編譯及安裝：
# cd /usr/local/src
# wget "https://git-core.googlecode.com/files/git-1.8.2.1.tar.gz"
# bunzip2 git-1.7.4.tar.bz2
# tar xf git-1.7.4.tar
# rm git-1.7.4.tar
# ln -s git-1.7.4 git
# cd git/
# make clean
# ./configure --with-curl --with-openssl --with-expat
# make
# make install
