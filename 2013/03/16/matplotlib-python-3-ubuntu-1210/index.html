安裝 matplotlib (Python 3) (Ubuntu 12.10)

由下列指令安裝：
$ sudo apt-get install python3-dev python3-pip python3-setuptools libfreetype6-dev libpng12-dev python3-pyqt4
$ sudo pip-3.2 install matplotlib

在要使用 matplotlib 的程式前加入：
import matplotlib
matplotlib.use('Qt4Agg')


如果不想每次寫程式都多那兩行
可以把這個設定檔
http://matplotlib.org/users/customizing.html#matplotlibrc-sample
放到這個路徑
~/.matplotlib/matplotlibrc

再將裡面的
backend      : GTKAgg
修改為
backend      : Qt4Agg
