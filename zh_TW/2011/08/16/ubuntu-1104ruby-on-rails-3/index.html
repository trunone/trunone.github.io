在Ubuntu 11.04安裝Ruby on Rails 3環境

雖然Ubuntu下可以很方便的直接安裝rails這個套件

來完成整個RoR環境的安裝 但是它的Rails卻是不是最新的3.0.x

因此我們要用手動的方法來安裝

首先安裝完整版的 Ruby 套件：
$ sudo apt-get install ruby-full

接下來安裝 RubyGems：
cd /tmp/
wget http://production.cf.rubygems.org/rubygems/rubygems-1.8.8.tgz
tar xvfz rubygems-1.8.8.tgz
cd rubygems-1.8.8
sudo ruby setup.rb
sudo gem update --system #Update RubyGems to the newest
sudo ln -s /usr/bin/gem1.8 /usr/bin/gem


最後就是安裝 Rails：
$ sudo gem install rails

RoR預設使用的資料庫是Sqlite 要用RubyGems安裝前 要先在Ubutnu裝好它的開發套件
$ sudo apt-get install libsqlite3-dev
