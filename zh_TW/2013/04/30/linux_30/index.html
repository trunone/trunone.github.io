設定 Linux 函式庫路徑

在 Linux 下增加程式讀取函式庫路徑的方法

當執行時：
% LD_LIBRARY_PATH=/path/to/custom/lib Command

新增到環境變數：
% export LD_LIBRARY_PATH=/path/to/custom/lib

新增到系統設定中：
% sudo ldconfig /path/to/custom/lib/
