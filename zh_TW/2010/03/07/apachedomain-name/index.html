apache啟動時抓不到Domain Name的問題

之前Ubuntu上的apache在啟動時老是出現這個錯誤訊息

apache2: Could not reliably determine the server's fully qualified domain name, using 127.0.0.1 for ServerName

雖然好像沒什麼影響 但是看了總是很礙眼 於是google了一下

才發現原來是apache設定的問題 只要在/etc/apache2/apache2.conf中加入
ServerName "server.name"
那個server.name就任你改囉

Reference: Apache error - server's fully qualified domain name [RESOLVED]
