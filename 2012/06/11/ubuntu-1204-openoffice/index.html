換裝 OpenOffice 3.4 (Ubuntu 12.04)

下載 OpenOffice 3.4: http://www.openoffice.org/download/index.html

移除 LibreOffice :
$ sudo apt-get purge libreoffice*

解壓縮 OpenOffice 3.4:
$ tar zxvf FILENAME

安裝 OpenOffice 3.4:
$ cd INSTALL_FOLDER/DEBS
$ sudo dpkg -i *.deb
$ cd desktop-integration
$ sudo dpkg -i *.deb
