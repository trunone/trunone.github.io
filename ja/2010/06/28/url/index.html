URL編碼/解碼

有時候複製的網址裡有包含ASCII以外的字元時

就會被換成%XX%XX……一類的編碼

把下面的HTML原始瑪存起來 便可以用來轉換

<html>
<head>
<title>URL Decode/Encode</title>
</head>
<body>
<center>
<font size=+2>URL Decode/Encode</font><br />
URL to Decode:
<TEXTAREA ID="String1" ROWS="10" COLS="30"></TEXTAREA>
URL Decoded:
<TEXTAREA ID="String2" ROWS="10" COLS="30"></TEXTAREA><br />
URL to Encode:
<TEXTAREA ID="String3" ROWS="10" COLS="30"></TEXTAREA>
URL Encoded:
<TEXTAREA ID="String4" ROWS="10" COLS="30"></TEXTAREA><br />
<INPUT TYPE="button" ID="Decode" value="Decode" onClick="javascript:String2.value=decodeURI(String1.value);">
<INPUT TYPE="button" ID="Encode" value="Encode" onClick="javascript:String4.value=encodeURI(String3.value);">
</center>
</body>
</html>
