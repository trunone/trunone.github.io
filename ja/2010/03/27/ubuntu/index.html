Ubuntu 升級發行版

升級的方法主要有兩種 一個是在X Window下 按 Ctrl+Alt+T 輸入：

$ upgrade-manager -d

在出現的升級管理員中應該就有新版本可以用囉

另一個是使用Shell 先確認是否有裝 update-manager-core 這個套件

再到 /etc/update-manager/release-upgrades 這個檔案確認

set Prompt 的部分 一般版要為 normal 長期支援版為 lts

接下來到Shell下輸入

$ sudo do-release-upgrade -d
