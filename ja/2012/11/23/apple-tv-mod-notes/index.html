網樂通改機雜記

想玩的 Professional 一點的話 就要從機板上接 UART 出來看 Console 的訊息

這裡要注意的是 這個 UART 是 TTL 的訊號 而不是 RS-232 的 所以買 USB 轉接線的時候要注意

在按任意鍵中斷 uboot 開機程序後 要改變 bootargs 來進入 shell

setenv bootargs console=ttyAS0,115200 root=8:2 rw rootfstype=ext3 \
rw rootflags=data=journal nwhwconf=device:eth0,hwaddr:10:08:E2:12:06:BD \
phyaddr:0,watchdog:5000 mem=120M bigphysarea=2048 init=/bin/sh

輸入 boot 開機完後 再到 /etc/inittabBB 將 ttyAS0 那行的註解拿掉

這樣每次開機就都有 Console 了
